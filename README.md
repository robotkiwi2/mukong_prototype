# 무공 (武功) — 타이핑 무협 액션 프로토타입

이동·방어·공격 등 모든 행위를 **타이핑(+엔터)으로 시전**하는 웹 기반 실시간 무협 액션 게임.
구결을 외워 무공을 펼치는 것이 핵심 판타지 — 타이핑 행위 자체가 세계관 내 무공 시전이다.

## 실행

브라우저는 `file://`에서 모듈·JSON 로드를 막으므로 **로컬 서버**로 실행한다.

```bash
# 프로젝트 폴더에서
python -m http.server 8000
# 또는: npx serve
```

브라우저에서 `http://localhost:8000` 접속.

## 조작 (프로토타입)

- 맵에서 아레나(취선루 / 뒷골목) 선택 → 진입
- 모든 행위는 **구결 입력 후 Enter** (우측 패널에 가능한 명령·설명)
  - 이동: `앞` `뒤` `좌` `우` `접근`
  - 공격: `주먹` `차기` …
  - 방어: `피` `굴러` `숙여`

## 구조

```
index.html · styles.css        앱 셸 / 화면
src/        data.js            데이터 로드 · 엔티티 파생 · 장소 병합 · 입력 판정
            arena.js           실시간 전투(렌더 · 적 AI · 입력)
            input.js           한글 IME 안전 입력
            sound.js           효과음(WebAudio)
            main.js            화면 상태기계 · 맵 · HUD
data/       *.json             캠페인 데이터 (장소·장면·인물·무공·병기·기본행위)
```

- **엔진 = 코드 / 캠페인 = 데이터.** 새 장소·전투는 데이터 추가로 확장.
